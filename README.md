🤖 Creatix

### Autonomous AI Coding Assistant

Creatix is an AI-powered coding assistant that helps developers generate code, explain code, debug errors, review GitHub repositories, and answer repository-related questions using RAG.

## ✨ Features

- 💻 Generate Code
- 📖 Explain Code
- 🐞 Debug Code
- 📂 Review GitHub Repositories
- 🔍 Repository Q&A (RAG)

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **Backend:** FastAPI
- **LLM:** Groq (Llama 3.3) / Google Gemini
- **Frameworks:** LangChain, LangGraph
- **Vector Database:** ChromaDB
- **API:** GitHub API
- **Language:** Python
